# nk_ReactTs_template

*Starter pack for React.Ts with axios*

## You can simply copy these files & type these commands

```
npm i
```
```
npm run dev
```

## If you want to do the complete installation type :
```
	npm create vite@latest
```

- enter project name

- same name for package

- select React

- select Typescript + SWC

```
	cd "project-name"
```
```
	npm i
```
```
	npm run dev
```
